Open an SSH client.

Locate your private key file. The key used to launch this instance is peer_v.pem

Run this command, if necessary, to ensure your key is not publicly viewable.
chmod 400 "peere_vkeey.pem"

Connect to your instance using its Public DNS:
ec2

Example:

ssh -i "peere_vkeey.pem" ec2-user@ec2
